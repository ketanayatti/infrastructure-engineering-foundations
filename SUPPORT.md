# Support

This repository is a long-term operational engineering knowledge system. Support focuses on clarifying technical understanding and improving the system.

## How to Ask Questions

- Be specific about the module, goal, and context.
- Include what you tried, what you expected, and what you observed.
- Provide logs or command output when relevant, with secrets removed.

## Where to Ask

- Use issues for actionable problems, corrections, or missing content.
- Use discussions for broader questions and design conversations if enabled.

## How to Contribute

If you can improve the repository, open a pull request with a clear summary and verification steps. Small, focused changes are preferred.

## Learning-first Philosophy

This project values deep, operational learning. Share your hypotheses, test results, and lessons learned. Evidence-based contributions are preferred over opinion-only changes.
