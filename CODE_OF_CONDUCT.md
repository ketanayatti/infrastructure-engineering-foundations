# Code of Conduct

This project is a professional, open-source engineering space focused on operational systems. We expect respectful, constructive collaboration.

## Expected Behavior

- Be respectful, direct, and professional.
- Provide evidence and context when making claims.
- Give feedback that improves the work, not the person.
- Assume good intent and seek clarity when needed.

## Unacceptable Behavior

- Harassment, discrimination, or personal attacks.
- Doxxing or sharing private information.
- Disruptive behavior that derails technical discussions.
- Repeatedly ignoring maintainers or project guidelines.

## Reporting and Enforcement

If you experience or witness unacceptable behavior, contact the maintainers via their GitHub profiles. Reports will be handled privately and fairly.

Maintainers may take any action they deem appropriate, including content removal or participation restrictions, to protect the integrity of the community.

## Scope

This Code of Conduct applies to all repository spaces, issues, pull requests, discussions, and related community interactions.
